# User Management System

A Spring Boot User Management System built as a backend portfolio project.

## Tech Stack

* Java
* Spring Boot
* Spring Security
* Thymeleaf
* MySQL
* Maven

## Features

### Authentication

* User Register
* User Login
* Password Encryption (BCrypt)

### Authorization

* ROLE_USER
* ROLE_ADMIN
* ROLE_SUPER_ADMIN

### User Management

* Profile page
* Change password
* Upload avatar

### Security

* Spring Security authentication
* Role-based access control

## How to Run

Clone the project

```
git clone https://github.com/Nkis9999/user-management-system.git
```

Run Spring Boot

```
mvn spring-boot:run
```

Open browser

```
http://localhost:8080
```
