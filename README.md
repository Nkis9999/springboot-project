# Spring Boot Member Management System

## Description

A member management system built with Spring Boot and MySQL.

Features:

- Member Registration
- Member Login
- Secure Password Storage
- MVC Architecture

## Architecture

Client
 ↓
Controller
 ↓
Service
 ↓
Repository
 ↓
Database

## Technology Stack

Backend:
- Java
- Spring Boot
- Spring Security
- Spring Data JPA
- Hibernate

Frontend:
- Thymeleaf
- Bootstrap

Database:
- MySQL

## Installation

1. Clone

git clone your-project-url

2. Create Database

CREATE DATABASE course;

3. Run Project

http://localhost:8080/login

## Security

Password encryption using BCryptPasswordEncoder.
